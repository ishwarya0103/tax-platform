# Tax Platform

Frontend-only prototype of an AI-powered tax platform for a CPA firm — individual and business
tax returns, AI-assisted document extraction, and CPA/client collaboration. Everything is mocked
with hardcoded TypeScript data; there is no backend.

## Stack

- Vite
- React + TypeScript
- Tailwind CSS v4
- React Router

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
